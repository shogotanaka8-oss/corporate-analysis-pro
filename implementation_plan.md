# 企業分析ツール 構築プラン

画像の「企業分析レポート」をWebアプリとして再現し、入力データの保存・管理ができるツールを構築する。

## Proposed Changes

### [Web Application Component]

#### [MODIFY] index.html
- 各入力項目のラベル付近に、計算式（例：ROE = 当期純利益 / 自己資本）を小さなグレー文字で追加。
- 保存成功時やエラー時を通知するための通知用トースト（ポップアップ）要素を追加。

#### [MODIFY] app.js
- **保存ボタンの改善**: クリック時に確実に `localStorage` への書き込みを行い、視覚的なポップアップを表示。
- **自動保存 (Auto-save)**: 入力値が変更されるたびに「下書き」としてセッションまたは `localStorage` に保存し、リロード時に復元できるようにする。
- **編集再開**: 履歴からクリックした際に、現在の編集内容を上書き保存または新規保存するか選べるようにロジックを微調整。

## Verification Plan

### Automated Tests
- なし（小規模なフロントエンドのみのため）

### Manual Verification
1. フォームにデータを入力し、「保存」ボタンを押す。
2. ページをリロードしてもデータが「履歴」に残っていることを確認。
3. 履歴からデータを選択し、入力フォームに正しく復元されることを確認。
4. スマホ・PC両方でレイアウトが崩れないか確認。
