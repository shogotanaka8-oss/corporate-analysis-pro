# 企業分析ツール 構築プラン

画像の「企業分析レポート」をWebアプリとして再現し、入力データの保存・管理ができるツールを構築する。

## Proposed Changes

### [Web Application Component]

#### [MODIFY] index.html
- **スマホ用タブナビゲーション**: 画面下部またはセクション上部に、スマホ時に表示されるタブ（財務、指標、定性、格付け）を追加。
- **データ共有機能**: 設定セクションに「データをコピー（エクスポート）」と「データを貼り付け（インポート）」ボタンを追加。

#### [MODIFY] index.css
- **スマホUIの強化**: 5ヶ年テーブルを横スクロールではなく、スマホで見やすいカード形式に変更するか、カラムを絞る表示を選択できるように調整。
- タブ切り替えのアニメーションとレイアウト。

#### [MODIFY] app.js
- **エクスポート/インポートロジック**: `localStorage` の全データを JSON 文字列としてクリップボードへコピー、または入力された JSON を読み込む処理を実装。
- タブの表示切り替え制御ロジック。

## Verification Plan

### Automated Tests
- なし（小規模なフロントエンドのみのため）

### Manual Verification
1. フォームにデータを入力し、「保存」ボタンを押す。
2. ページをリロードしてもデータが「履歴」に残っていることを確認。
3. 履歴からデータを選択し、入力フォームに正しく復元されることを確認。
4. スマホ・PC両方でレイアウトが崩れないか確認。
