# 企業分析ツール 構築プラン

画像の「企業分析レポート」をWebアプリとして再現し、入力データの保存・管理ができるツールを構築する。

## Proposed Changes

### [Web Application Component]

#### [NEW] index.html
- フォームの構造を定義。画像にある全項目（健全性、収益性、成長性、株価指標、バリューチェック、投資判定）を適切にセクション分けして配置。
- 各入力項目は「項目名」と「数値入力」を明確に分け、使いやすさを重視。

#### [NEW] index.css
- 「プレミアムなデザイン」を実現するためのスタイル。
- シックな色使い（ダークモード対応も検討）や、カードレイアウト、美しいタイポグラフィを採用。

#### [NEW] app.js
- フォーム入力内容を JSON 形式でまとめ、`localStorage` に保存。
- 過去の分析結果を「履歴」として一覧表示し、再読み込みや削除ができる機能を実装。

## Verification Plan

### Automated Tests
- なし（小規模なフロントエンドのみのため）

### Manual Verification
1. フォームにデータを入力し、「保存」ボタンを押す。
2. ページをリロードしてもデータが「履歴」に残っていることを確認。
3. 履歴からデータを選択し、入力フォームに正しく復元されることを確認。
4. スマホ・PC両方でレイアウトが崩れないか確認。
