# 企業分析ツール 構築プラン

画像の「企業分析レポート」をWebアプリとして再現し、入力データの保存・管理ができるツールを構築する。

## Proposed Changes

### [Web Application Component]

#### [MODIFY] index.html
- 全体的なレイアウトをマルチカラムまたはスクロール可能なセクションに拡張。
- **5年間の財務推移テーブル**: 項目（自己資本比率、ROE等）ごとに5年分の数値を入力できるテーブルを追加。
- **詳細株価指標セクション**: 近似PER、EV/EBITDA、ROICなどの専門的な指標入力欄を追加。
- **定性・企業価値セクション**: 大株主リストや配当方針、清算価値などを入力するテキストエリア・入力欄を追加。
- **総合格付けセクション**: 各評価項目（A〜E）と総括を入力するドロップダウン/テキストボックスを追加。

#### [MODIFY] index.css
- 大量の入力項目を整理するためのスタイル調整。
- 入力テーブルのレスポンシブ対応。
- セクションごとの視認性を高めるためのアクセントカラーの活用。

#### [MODIFY] app.js
- より複雑になったデータ構造を `localStorage` に保存・復元できるよう保存ロジックを最適化。
- テーブルデータの動的な取得と反映。

## Verification Plan

### Automated Tests
- なし（小規模なフロントエンドのみのため）

### Manual Verification
1. フォームにデータを入力し、「保存」ボタンを押す。
2. ページをリロードしてもデータが「履歴」に残っていることを確認。
3. 履歴からデータを選択し、入力フォームに正しく復元されることを確認。
4. スマホ・PC両方でレイアウトが崩れないか確認。
