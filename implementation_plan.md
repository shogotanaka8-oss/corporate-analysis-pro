# 企業分析ツール 構築プラン

画像の「企業分析レポート」をWebアプリとして再現し、入力データの保存・管理ができるツールを構築する。

## Proposed Changes

### [Web Application Component]

#### [MODIFY] index.html
- **生データ入力テーブルを追加**: 財務指標の「原料」となる数値（売上高、営業利益、純利益、総資産、自己資本、流動資産、流動負債）を入力する専用テーブルをタブ「財務」内に追加。
- **指標テーブルの自動計算化**: 上記の生データが入力された際、リアルタイムで各指標（自己資本比率、ROE等）が更新されるように変更。

#### [MODIFY] app.js
- **自動計算エンジン**: 生データから指標を算出する専用のユーティリティ関数を実装。
- **リアクティブ・リスナー**: 生データの入力フィールドに `input` イベントを貼り、変更を即座に計算結果へ反映。
- **データ構造の拡張**: 生データそのものも `localStorage` に保存し、再編集時に指標を再計算できるように調整。

## Verification Plan

### Automated Tests
- なし（小規模なフロントエンドのみのため）

### Manual Verification
1. フォームにデータを入力し、「保存」ボタンを押す。
2. ページをリロードしてもデータが「履歴」に残っていることを確認。
3. 履歴からデータを選択し、入力フォームに正しく復元されることを確認。
4. スマホ・PC両方でレイアウトが崩れないか確認。
